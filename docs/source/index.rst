Welcome to argoPlot-script documentation!
=========================================

**argoPlot-script** is a Python library for download and visualize the argoFloat that tracking the trajectories and its parameters variability.
It pulls data from the `argoFloat database <https://argo.ucsd.edu/data/>`_
and offers a *simple* and *intuitive* visualization.

The argoPlot-script has its documentation and this project is under active development.

Check out the sections for further information, including
how to download and visualize the data.

.. note::

   This project is under active development.
